# 旅游信息管理系统 python229

## 项目概述

旅游信息管理系统是基于Python与Vue开发的一套旅游推荐系统。该系统结合了前端Vue的响应式设计与后端Python的高效数据处理能力，通过Django框架与Scrapy爬虫技术，为用户提供一站式的旅游信息服务。

## 技术栈

- 前端：Vue
- 后端：Python、Django、Scrapy
- 数据库：MySQL

## 功能模块

### 用户模块

- 登录
- 注册
- 首页
- 景点信息
- 热门景点信息
- 特色美食
- 游玩路线
- 个人中心
- 投诉建议
- 旅游资讯

### 管理员模块

- 用户管理
- 景点信息管理
- 热门景点管理
- 特色美食管理
- 游玩路线管理
- 基础数据类型管理
- 轮播图管理
- 个人中心
- 资讯管理

### 员工模块

- 活动查看
- 订单管理
- 个人中心

## 系统角色

- 用户：普通用户，可访问用户模块的功能。
- 管理员：负责管理后台，对网站内容进行维护。
- 员工：负责处理日常运营事务。

## 运行环境

- 后端：Python 3.x，Django 2.x，Scrapy 1.x
- 前端：Vue.js，Node.js
- 数据库：MySQL 5.x 或更高版本

## 部署步骤

1. 配置后端开发环境，安装Python、Django、Scrapy等必要的库。
2. 配置数据库MySQL，导入初始数据。
3. 部署前端Vue项目，构建静态资源。
4. 将构建后的前端资源放置于后端项目指定的静态文件目录。
5. 启动后端Django服务。
